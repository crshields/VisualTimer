# Visual Countdown Timer
A minimalist color and circle-completion based countdown timer designed for classroom use. 

# Why numberless?
Watching a number count down causes anxiety for many students and can distract those that are prone to staring at moving or quickly changing objects. This timer is meant to provide the percentage of time remaining at a glance, but to be too-slow changing to inspire distraction. 

# How To Use
Enter a duriation in the time field and click **Start**
Accepted formats: 
- `25:00` — mm:ss
- `01:20:00` — hh:mm:ss
- `80m`, `45s`, `1h 20m` — plain language
- `90` — interpreted as minutes
- Clicking the ring itself will cause the timer to pause/resume. Use **Fullscreen** to fill a classroom display.

- #Features
- No numbers displayed on the ring
- Color shifts green to red as time advances
- Works entirely in the browser - no install, no account, no internet required after downloading the page.
